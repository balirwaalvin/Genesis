## Color Scheme

### Primary Colors:
- **Background**: Dark blue-gray gradient (#0f172a â†’ #1a1f35)
- **Primary Accent**: Blue gradient (#6366f1 â†’ #ec4899)
- **Text Primary**: White/near-white
- **Text Secondary**: Gray (#94a3b8)

### UI Elements:
- **Cards/Bubbles**: Semi-transparent dark (#1e293b with opacity)
- **Borders**: Subtle blue glow (rgba(99, 102, 241, 0.3))
- **Hover States**: Brighter blue glow
- **Send Button**: Gradient background with glow

## Key Animations

1. **Page Load**: Fade in with slight delay
2. **Messages**: Slide up + fade in
3. **Typing Indicator**: Bouncing dots
4. **Buttons**: Scale on hover (1.05x) and tap (0.95x)
5. **Dropdown**: Fade + scale animation
6. **Input Focus**: Border glow intensifies

## Responsive Breakpoints

- **Desktop**: > 768px
  - Full layout with side padding
  - 900px max-width for chat content
  - Large suggestion cards (2x2 grid)

- **Mobile**: â‰¤ 768px
  - Compact header
  - Single column suggestion cards
  - Smaller fonts
  - Touch-optimized buttons

## Typography Scale

- **Logo**: 20px (mobile: 18px)
- **Welcome Title**: 32px (mobile: 24px)
- **Messages**: 15px (mobile: 14px)
- **Input**: 15px
- **Disclaimer**: 12px
- **Model Selector**: 13px

## Spacing System

- **Small**: 8px (gaps, padding)
- **Medium**: 16px (card padding)
- **Large**: 24px (section padding)
- **XLarge**: 40px (welcome section)

## Interactive States

### Buttons:
- **Default**: Semi-transparent background
- **Hover**: Lighter background + glow
- **Active**: Slight scale down
- **Disabled**: Reduced opacity + no interaction

### Input:
- **Default**: Subtle border
- **Focus**: Bright border + outer glow
- **Disabled**: Reduced opacity

### Messages:
- **AI**: Left-aligned, gradient avatar
- **User**: Right-aligned, simple avatar
- **Actions**: Hidden by default, show on hover

---

This layout provides a clean, modern, and distraction-free interface focused entirely on the chat experience, just like Gemini and ChatGPT.
